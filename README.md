# About the Game

- This is a little Hardcoded game i did myself, coded in C# with visual Studios. Game runs on no Engine.

- Dungeon Hunter is a single-player fantasy RPG where you start by surviving a brutal dungeon and slowly uncover that the world above it is just as dangerous. You fight monsters, bosses, and hidden endgame threats, collect gear, level classes and abilities, tame companions, and build your strength across both dungeon runs and overworld progression.

  As the game opens up, it becomes more than a crawler. You can travel between kingdoms, build reputation, join armies, uncover hidden factions, buy homes, create your own kingdom, explore secret realms like Evermore, the Void Force, and the Demon Realm, and shape how the wider world changes through war, politics, and late-game choices.

  It is best described as a progression-heavy console RPG that mixes dungeon crawling, world lore, kingdom politics, hidden routes, and long-term character growth.
